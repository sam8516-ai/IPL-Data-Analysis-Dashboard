# IPL Data Analysis Dashboard

## Overview

This project is an interactive IPL Data Analysis Dashboard built using Power BI. It provides comprehensive insights into Indian Premier League (IPL) performance, player statistics, team records, and season-wise trends from 2008 to 2025 through dynamic visualizations and interactive filters.

## Key Features

### Season-wise Analysis

* Dynamic season slicer for IPL seasons (2008–2025)
* Automatic update of all visuals and KPIs based on selected season

### Batting Analysis

* Orange Cap Winners
* Top Run Scorers
* Total Runs Analysis
* Centuries and Half-Centuries Statistics
* Boundary Analysis (Fours and Sixes)

### Bowling Analysis

* Purple Cap Winners
* Top Wicket Takers
* Economy Rate Insights
* Bowling Performance Metrics

### Team Performance Analysis

* Team Standings
* Matches Played
* Wins and Losses
* Win Percentage Analysis

### Interactive Dashboard Features

* Dynamic Player Profile Section
* URL-based Player Image Integration
* Interactive Charts and KPI Cards
* Cross-filtering and Drill-down Analysis

## Tools and Technologies

## Insights Derived

* Identified the highest-performing teams across different IPL seasons.
* Analyzed Orange Cap and Purple Cap winners season-wise.
* Compared player performances using batting and bowling metrics.
* Discovered trends in team success rates and win percentages.
* Evaluated boundary-hitting patterns through fours and sixes analysis.
* Tracked season-wise performance changes of teams and players.

## How to Use

1. Download the Power BI dashboard file.
2. Open the `.pbix` file using Microsoft Power BI Desktop.
3. Use season slicers and filters to explore the data.
4. Interact with visualizations to analyze team and player performance.
5. Gain insights from dynamic charts, KPIs, and player profiles.
